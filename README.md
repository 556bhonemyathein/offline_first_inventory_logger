# Offline First Inventory Logger

## Features
- Fetch suppliers from API
- Add inventory item
- Edit inventory item
- Delete inventory item
- Local SQLite storage
- Riverpod state management

## Packages Used
- flutter_riverpod
- dio
- sqflite
- flutter_slidable
- intl
- logger
- path
- flutter_slidable

## Run Project

- flutter pub get
- flutter run

## Architecture
- MVC-like clean structure
- UI -> Provider -> Repository -> Service -> Database/API
- Riverpod for state management
